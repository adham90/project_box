# Setup the environment

This application requires:

- Ruby [2.2.\*, 2.3.*]
- Rails 5.0.0
- Mongodb

**Using RVM**

If you're using RVM then you might want to run

> $ rvm use --create --ruby-version 2.3.1@project_box

### Run test.

> $ rspec
